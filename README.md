#### 项目介绍与注意事项： 
本项目用于合成三方数字人短视频，可用于heygen的部分平替（因为hg的审核越来越严格）。
输入源视频+音频，输出口型匹配的高清视频。

#### 实现流程原理
运用Wav2lip将音频与视频进行口型同步合成，再运用GFP-GAN进行超分提升画质
